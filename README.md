# Connect4TaskOvarro
Repository for the task given by Ovarro

I chose to code this in Java as it is one of the languages I am more proficient in, and something Ovarro is looking for. It was coded inside Intelli J however using the maven build tool 
instead of the standard Intelli J one for running tests, although I did not manage to be able to create some tests.

To run the code either clone the repo and run it in an IDE with Maven and Java installed, or run the jar file.

##how to run the jar file
java -jar connect4.jar

in the folder directory
